# Notes Tool  

Command line notes tool.

## Get started  
Client:  
```  
pipenv shell  
python notesClient.py <cmd>  
```
Backend: 
```  
docker-compose up --build  
```  

## Shell alias  
Create alias in ~/.zshrc or ~/.bashrc
```
alias note='python notesClient.py'
```

## Build with  
* Docker  
* MongoDB  
* NodeJs / express
* Python 3.7.0  

### Ports  
* Mongo-express: 8081
* REST API: 3000
